AI Recruiter Backend API

A modern, AI-powered recruitment platform for recruiters and job applicants.
API Status: Online
View Interactive API Docs (Swagger)

Key Features

  • 🔒 Secure JWT Authentication
  • 🧑‍💼 Recruiter & Applicant Workflows
  • 🏢 Company & Team Management
  • 📄 Resume & Cover Letter Generation (AI)
  • 📊 Analytics & Dashboard
  • 💬 AI Chat Assistant
  • 📁 File Upload & Management
  • 🛡️ Rate Limiting & Security

Popular Endpoints

  • POST /api/v1/auth/register – Register user (with company support)
  • POST /api/v1/auth/login – User login
  • GET /api/v1/companies – List companies
  • POST /api/v1/companies – Create company
  • GET /api/v1/dashboard/jobs – Recruiter jobs
  • POST /api/v1/applicant/ai/generate-resume – AI resume generation
  • POST /api/v1/chat/message – AI chat assistant
